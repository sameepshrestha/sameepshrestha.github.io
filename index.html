Robotics Projects

MR1 Robot

MR1 robot was able to navigate through complex path, was semi automatic, and it also had the ability to pass an rectangular block to the other robots effotlessly without any time. It also had a grapper to hold objects of definite shape, to extend and to launch the object which can be used in different industrial scenario. All the graplers present in the robot was actuated using pneumatic actuators.

MR2 Robot

MR2 robot faced different design changes over the course of the year. The lack of high torque actuators really caused problems to make a fully functional quadruped. This design has a under actuated legs and is based on four bar mechanism. The robot is also equipped with a ratchet grappler to grap the object passed by the MR1 robot. The robot completed its task without problem, passing through tough terrains of robes, inclined plane etc.
